select-return

Return a function based on a condition. Functional alternative to switch-case.

Usage

const a = 100;

const result = new Select(a)
    .for(x => x>10,
        a => `${a} is greater than 10.`)
    .for(x => x<10,
        a => `${a} is lesser than 10.`)
    .for(x => x===10,
        a => `${a} is 10.`)

console.log(result.resolve()); // -> "100 is greater than 10."
